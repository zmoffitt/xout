# Sabre file PII scrubber

## To install

1. Make sure you have python 3 installed (like `3.6.2` or `3.7`):

```
	python --version
```

2. Install the [pipenv](https://pipenv.readthedocs.io/en/latest/) package manager.

On MacOS:

```
	brew install pipenv
```

3. Install xout

```
pip3 install . --user
```
